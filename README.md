# flask-QnA
QnA sites flask and python 

## 실행 방법
### 1. download source

### 2. 기본 python & Flask 설치
```
> # install python 3.8.5
> python -V 
python 3.8.5
> pip install Flask
> python -m pip install --upgrade pip
```

### 3. Flask 실행
```
> set FLASK_APP=pybo
> set FLASK_ENV=development
> flask run
```
- http://127.0.0.1:5000 접속
